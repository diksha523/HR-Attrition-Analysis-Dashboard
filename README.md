# HR Attrition Analysis Dashboard
# Project Overview

This project presents an interactive HR Attrition Analysis Dashboard developed using Power BI. The dashboard helps organizations understand employee turnover patterns and identify key factors contributing to attrition.

# Objectives
Analyze employee attrition trends
Understand department-wise attrition
Evaluate employee demographics
Monitor job satisfaction levels
Support data-driven HR decisions

# Tools Used
Power BI
DAX
Power Query
Microsoft Excel / CSV

# Key Insights
Attrition Rate Analysis
Department-wise Employee Distribution
Gender Analysis
Age Group Analysis
Job Satisfaction Analysis
Education Field Analysis

# How to Use
Download the .pbix file
Open using Power BI Desktop
Refresh data if required
Explore interactive visualizations
