# HR Attrition Analysis Dashboard

## Project Overview

This project presents an interactive HR Attrition Analysis Dashboard developed using Power BI. The dashboard helps organizations understand employee turnover patterns and identify key factors contributing to attrition. The dashboard provides valuable insights into workforce trends, employee demographics, and job satisfaction, enabling data-driven HR decision-making.

---

## Objectives

- Analyze employee attrition trends
- Understand department-wise attrition
- Evaluate employee demographics
- Monitor job satisfaction levels
- Support data-driven HR decisions

---

## Tools Used

- Power BI
- DAX (Data Analysis Expressions)
- Power Query
- Microsoft Excel / CSV

---

## Key Insights

- Attrition Rate Analysis
- Department-wise Employee Distribution
- Gender-wise Attrition Analysis
- Age Group Analysis
- Job Satisfaction Analysis
- Education Field Analysis
- Employee Count by Department
- Workforce Demographics Overview

---

## Features

- Interactive dashboard with filters and slicers
- Dynamic KPI cards for quick insights
- Department-wise attrition comparison
- Gender and age-based employee analysis
- Job satisfaction and education field analysis
- Easy-to-understand visual reports

---

## How to Use

1. Download the `.pbix` file from this repository.
2. Open the file using **Power BI Desktop**.
3. Refresh the dataset if required.
4. Explore the dashboard using filters and interactive visualizations.

---

## Dataset

The dataset contains employee-related information including:

- Employee Demographics
- Department Details
- Job Roles
- Education Fields
- Job Satisfaction Ratings
- Attrition Status

---

## Business Benefits

- Identifies major factors contributing to employee attrition.
- Helps HR teams improve employee retention strategies.
- Supports workforce planning and decision-making.
- Provides actionable insights through interactive visualizations.
